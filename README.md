## Instructions

Use this as a practice repo to simply add content to and then structure the content in the `index.html` page.

**1. Add the Unstructured Content**

- Copy the content from `content.txt` and paste it into the `<body>` section of `index.html`

**2. Structure the Content**

- Add `The Ithaca Zoo` in `<title>` in the `<head>` section
- Add structure, markup, the content in the `<body>` section:
  - add a main heading `<h1>`
  - add a sub heading `<h2>`
  - add 2 paragraphs `<p>`
  - make the advisory message `<strong>`

**3. Bonus - Add more structure to the content:**

  - group the `<h1>` into a `<header>`
  - group the `<h2>` and 2 `<p>'s` into a `<section>`
  - add a `<footer>` to the bottom of the page with the content
    `&copy; 2026 Ithaca Zoo`
